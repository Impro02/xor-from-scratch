## C++ Implementation

### Prerequisites

- CMake 3.10 or higher
- Eigen3
- fmt library

### Building the Project

1. Run CMake preset to configure the project:

   ```sh
   cmake --preset=default
   ```

2. Build the project:
   ```sh
   cmake --build build
   ```

### Running the Project

After building, you can run the executable:

```sh
./XorFromScratch
```
